# Typola 字体资源

`source-han-serif-sc-vf-subset.woff2` 来自 Adobe 官方 [Source Han Serif](https://github.com/adobe-fonts/source-han-serif) 2.003R 的 `SourceHanSerifSC-VF.ttf`，按 Typola UI 字符生成 WOFF2 子集。

- 字体族：`Source Han Serif SC VF`
- 许可：SIL Open Font License 1.1，见同目录 `OFL-1.1.txt`
- 未包含在子集中的字符继续沿 CSS 字体回退链渲染，不请求外部 CDN。
