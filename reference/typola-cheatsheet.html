右键 / Ctrl+K 动作

浮条只有 3 个,以下 4 个走菜单:

缩写
砍一半长度,保留核心信息
扩写
补充论据 / 例证,保持原风格立场
校对
改错别字、语法、标点
自定义
空模板,自己写要求(例如"翻成英文,保留技术名词")

SkillHub

场景 预置 skill
报告生成 空模板(等你手动加)
PPT 制作 huawei-style-ppt-skill · guizang-ppt-skill · huashu-slides · baoyu-slide-deck
HTML 制作 frontend-slides
SkillHub 规则
  • 系统预置不可删,用户添加可删
  • 未装的 skill → 卡片置灰 → 点击发"安装指令"给 AI 工作台当前对话
  • 每个 skill 一个独立 session UUID(互不串上下文)
  • Claude / OpenCode 看到的 skill 路径不一样(~/.claude/skills/ vs ~/.opencode/commands/)
