# Typola 使用教程 — 课程索引

> 目标读者:**已装好 Typola、要真用起来**的中文写作者。
> 心法:**文档是产物,AI 是工序。**
> 完成这 9 课后,你能在 30 分钟里走完一篇稿子的「写 → 改 → 检视 → 交付」。

## 学习路径图

```
[0 安装上手]
   │
   ▼
[1 浮条 AI 改稿] ── 选中一段原地对比
   │
   ▼
[2 AI 工作台开箱] ── 左栏对话框 + 多轮对话
   │      │
   │      ├─→ [3 上下文芯片] ── 喂 AI 素材
   │      │
   │      └─→ [6 多会话 Pill] ── 并行多任务
   │
   ▼
[4 Skill 场景] ── 报告/PPT/HTML 模板
   │      │
   │      └─→ [7 产物中心] ── 生成物回文件
   │
   ▼
[5 检视+审阅] ── 攒意见 → AI 改稿 → 逐处采纳
   │
   ▼
[8 交付+降级] ── PDF/Word/HTML/review.md + 无网兜底
```

## learning-records

每次课新增的"非显然"教训写 `learning-records/NNNN-<kebab>.md`。

## 每节课怎么读

1. 提前装好 CLI(第 0 课会指引)。
2. 每节课 5–20 分钟。开头有「一句话目标」,结尾有「试一试」+「陷阱」+「下一节预告」。
3. 永远可以问 AI 让它解释哪一步。

## 4 句贯穿全课程的口诀

> **Tab 管文件,Pill 管对话。**
> **左栏是草稿,中栏是成品。**
> **AI 改稿先 diff,采纳再回写。**
> **降级不离本地,产物总回文件。**
