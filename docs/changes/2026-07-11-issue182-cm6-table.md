# Issue #182 — CM6 表格增强

CM6 现支持 GFM 表格插入、当前列左/中/右对齐、行列插入/删除，以及 HTML、Excel/TSV、CSV 粘贴转 Markdown。所有操作都通过单个 `view.dispatch` 修改 source，进入原生 undo/redo。

表格单元格继续复用 `@atomic-editor/editor` 的 `tables()` widget；宽表保持在编辑器内部横向滚动，不撑破布局。右键菜单和 `Ctrl/Cmd+Alt+方向键` 提供行列操作入口。

范围不包括公式、多 sheet、复杂 HTML rowspan/colspan 保真或跨文件表格存储。

验证：`npm run typecheck`、`npm run lint`、`npm test`。

关联：#182，前置 #179，分析基线 #181。
